Scotiabank Statement Formats and What Gets Extracted

Scotiabank uses a consistent statement layout across most personal accounts, but Scotia Scene+ credit card statements include a rewards point column adjacent to each transaction amount — a column that generic converters often mistake for a dollar amount, producing incorrect transaction values in the output. Zera Books identifies and ignores reward columns, extracting only monetary transaction data.

Scotiabank business accounts, particularly those used by import/export businesses, frequently hold both CAD and USD account statements. When a client submits both in the same batch, Zera Books separates CAD and USD transactions by currency — producing output files with the currency code labelled per transaction, which is required for proper multi-currency accounting in QuickBooks Online Canada and Xero.
