RBC PDF Statement Format

RBC personal account statements use a clean table layout: Transaction Date, Description, Withdrawals, Deposits, Balance. The date format is typically month-abbreviated (e.g., "Jan 15, 2025").

RBC business account statements are more detailed. They include cheque images summary, PAD (Pre-Authorized Debit) details, and may span 10-20 pages for active accounts. Large-volume RBC business statements with 200+ transactions per month are where PDF conversion tools differ most — Zera Books processes all pages in a single pass.

RBC Business CSV Export

RBC's business CSV export includes two description columns (Description 1 and Description 2) and two separate currency columns (CAD$ and USD$). This is designed for multi-currency business accounts. Before importing into QuickBooks or Xero, you need to:

  • Combine Description 1 and Description 2 into a single description
  • Select only the currency column matching your accounting software account currency
  • Remove the Account Number and Account Type columns

Zera Books handles RBC's dual-currency format during conversion and outputs a single clean file per account/currency.

Frequently Asked Questions

How do I download RBC Royal Bank statements as PDF?

Log into RBC Online Banking, go to My Accounts, select the account, then click Statements. eStatements are available as PDFs for up to 7 years. You can also export transaction activity as CSV from the Transaction History page.

Does RBC Royal Bank work with QuickBooks Online Canada?

Yes. RBC supports direct bank feed connections to QuickBooks Online Canada. For historical statements or accounts without feed access, download the PDF and convert to QBO using Zera Books, then import via QuickBooks Banking.

What is the RBC CSV export format for business accounts?

RBC business account CSV exports use: Account Number, Account Type, Transaction Date, Cheque Number, Description 1, Description 2, CAD$, USD$. You need to combine Description 1 and Description 2 and select the correct currency column before importing into accounting software.
