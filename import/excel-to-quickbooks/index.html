Excel to QuickBooks: Import Guide for Bank Transactions

QuickBooks does not accept XLSX files. If you have bank transaction data in Excel — from a bank export, a reconciliation spreadsheet, or a client-provided file — you need to convert it to CSV with the right formatting before QuickBooks will accept it.

Step 1: Clean the Excel File

Before converting to CSV, clean the Excel file:

  1. Remove summary rows: Delete opening/closing balance rows, total rows, and blank spacer rows
  2. Remove extra columns: Keep only Date, Description, and Amount (or Debit/Credit). Extra columns like Running Balance will confuse QuickBooks' column mapper
  3. Fix date format: Select the date column > Format Cells > Custom > enter mm/dd/yyyy
  4. Fix amount format: Select amount cells > Format Cells > Number > 2 decimal places (uncheck "Use 1000 separator")
  5. Check negative signs: Withdrawals must be negative. If your bank export shows debits as positive, multiply the column by -1 for debits

Step 2: Save as CSV

  1. Go to File > Save As
  2. Change "Save as type" to CSV (Comma delimited)
  3. Click Save — Excel will warn about formatting loss, click Yes
  4. Open the CSV in Notepad to verify the output looks correct

Step 3: Import into QuickBooks

  1. In QuickBooks Online, go to Banking > Upload transactions
  2. Select the CSV file you saved
  3. Map columns: Date, Description, Amount
  4. Confirm date format matches your QuickBooks region
  5. Review the preview and click Next to import

Frequently Asked Questions

Can QuickBooks import Excel files directly?

No. QuickBooks Online does not accept XLSX files. You must save the Excel file as CSV first (File > Save As > CSV Comma delimited), then upload the CSV through QuickBooks Banking > Upload transactions.

Why do my Excel dates look wrong after importing to QuickBooks?

Excel stores dates as serial numbers internally. When you save as CSV, dates are written as text in whatever format Excel displays them. If Excel shows 15-Jan-25 instead of 01/15/2025, QuickBooks may reject or misread the date. Format the date column as MM/DD/YYYY before saving as CSV.

How do I remove currency formatting from Excel before QuickBooks import?

Select the Amount column, press Ctrl+1 to open Format Cells, choose Number with 2 decimal places (no currency symbol), then save as CSV. Alternatively use a formula like =VALUE(SUBSTITUTE(A2,"$","")) to strip the symbol.
