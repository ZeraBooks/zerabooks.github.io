Xero Bank Statement Import: Step-by-Step Guide

Xero accepts bank statement files in OFX, QIF, CSV, and OFC formats. The manual import option is useful when your bank doesn't have a direct Xero bank feed or when you need to import historical transactions not covered by the feed.

Recommended format: OFX gives you the cleanest import into Xero — no column mapping needed. If you have PDF statements, Zera Books converts them directly to OFX or Xero-formatted CSV in under 60 seconds.

Manual Import Steps

  1. In Xero, go to Accounting > Bank Accounts
  2. Click the bank account you want to import into
  3. Select Import a Statement (top right)
  4. Upload your OFX, QIF, or CSV file
  5. For CSV: map the columns (Date, Amount, Payee, Description, Reference)
  6. Review imported transactions in the Reconcile tab

CSV Column Requirements

Xero's CSV import is flexible but you must tell it which column is which during upload. The minimum required columns are:

ColumnNotes
DateMust match your Xero date format setting
AmountNegative for money out, positive for money in. Or use separate Spent/Received columns.
PayeeOptional but improves auto-matching
DescriptionMaps to transaction narration in Xero
ReferenceOptional — useful for cheque numbers or invoice refs

OFX Import

OFX files require no column mapping in Xero. The file contains structured transaction data including FITID (unique transaction ID), which Xero uses to prevent duplicates. If you import the same date range twice, Xero will skip transactions it has already seen.

Duplicate Handling

Xero uses the transaction date, amount, and description to flag potential duplicates. If your bank feed overlaps with a manual import, you will see duplicates flagged in the Reconcile tab. Delete the duplicate before coding the transaction.

Common Issues

  • File size limit: Xero caps manual imports at 5 MB. For larger files, split by month.
  • Password-protected PDFs: Must be unlocked before conversion. Zera Books handles password-protected files directly.
  • Multi-currency accounts: The currency in your import file must match the account's currency in Xero.

Frequently Asked Questions

What format does Xero use to import bank statements?

Xero accepts OFX, QIF, CSV, and OFC files for manual bank imports. OFX is the most reliable. CSV imports require four specific columns: Date, Amount, Payee, Description.

How do I import a PDF statement into Xero?

Xero does not read PDF files. Convert the PDF to OFX or CSV first using a converter like Zera Books, then upload via Accounting > Bank Accounts > Import a Statement.

Why is Xero showing wrong dates after CSV import?

Xero reads the date format set in your organisation's settings. If your CSV uses DD/MM/YYYY but Xero is set to MM/DD/YYYY, dates for the first 12 days of each month will swap. Match the date format in the file to your Xero regional setting.
