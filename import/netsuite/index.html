NetSuite Bank Statement Import: Step-by-Step Guide

NetSuite's Banking module handles bank statement imports through the Import Bank Data feature. It accepts OFX and CSV files. The imported transactions then go through the Reconcile Bank Statement workflow to match against your GL.

Import Steps in NetSuite

  1. Go to Transactions > Bank > Import Bank Data
  2. Select the bank account from the subsidiary/account dropdown
  3. Choose file type: OFX or CSV
  4. Upload your file and click Import
  5. NetSuite maps transactions and shows an import summary (matched, unmatched, errors)
  6. Go to Transactions > Bank > Reconcile Bank Statement to start reconciliation

CSV Format for NetSuite

ColumnFormatNotes
DateMM/DD/YYYYRequired
Transaction IDAlphanumericOptional but prevents duplicates
DescriptionTextRequired
Debit AmountPositive numberMoney out of the account
Credit AmountPositive numberMoney into the account

NetSuite's CSV import uses separate Debit and Credit columns — not a signed single Amount. If your CSV has a signed Amount column, you'll need to split it before importing. Leave the Debit cell blank for credits and vice versa.

OFX Import

OFX is the cleaner option. NetSuite reads the FITID field as a unique transaction ID to prevent duplicates. If your bank provides OFX downloads, use those. For banks that only provide PDF statements, convert to OFX first.

Matching Rules

NetSuite tries to auto-match imported bank transactions to existing GL entries based on amount and date (within a tolerance window you configure). Unmatched transactions appear in the reconciliation screen for manual matching or creating new GL entries.

Frequently Asked Questions

What file format does NetSuite use for bank statement imports?

NetSuite Bank Data (formerly NetSuite Bank Feeds) accepts OFX files. NetSuite also supports CSV import via the Banking module's Import Bank Data feature, or via SuiteScript for custom implementations.

How do I reconcile a bank statement in NetSuite?

In NetSuite, go to Transactions > Bank > Import Bank Data. Upload your OFX or CSV file, select the account, then use Reconcile Bank Statement to match imported transactions against your GL entries.

Can NetSuite import PDF bank statements?

NetSuite cannot read PDFs natively. Convert the PDF to OFX or CSV first using Zera Books, then import via the NetSuite Banking module. Zera Books exports NetSuite-compatible CSV with all required fields.
