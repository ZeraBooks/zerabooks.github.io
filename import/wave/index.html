Wave Accounting Bank Statement Import: Step-by-Step Guide

Wave supports manual CSV imports for bank transactions. Unlike QuickBooks or Xero, Wave does not accept OFX or QBO files — CSV is the only option for manual import outside of Wave's bank connection feature.

Import Steps

  1. Log into Wave and go to Accounting > Transactions
  2. Click More (top right) > Import transactions
  3. Select the account you want to import into
  4. Upload your CSV file
  5. Map columns: Date, Description, Amount (or Deposit/Withdrawal)
  6. Review the preview and confirm the import

CSV Format Requirements

Wave's CSV import accepts flexible column naming but the data types must be consistent:

ColumnFormatNotes
DateMM/DD/YYYY or YYYY-MM-DDMust be consistent throughout the file
DescriptionTextShown as transaction memo in Wave
AmountNegative for withdrawalsOr use separate Deposit/Withdrawal columns
DepositPositive onlyLeave blank for withdrawals
WithdrawalPositive onlyLeave blank for deposits

Wave does not require a header row but including one makes column mapping easier during import. Do not include totals rows, summary lines, or empty rows in the file — Wave will try to import them as transactions.

Tips for Clean Imports

  • Remove any footer rows (e.g., "Opening Balance", "Closing Balance") before importing
  • Check that amounts use a period as the decimal separator, not a comma (important for non-US statements)
  • Wave's category suggestions are basic — use Zera Books if you want transactions pre-categorized before import
  • Wave has no IIF export, so if you later move to QuickBooks, you'll need to re-import from your original statement files

Handling Duplicates

If a bank feed is already connected, importing a CSV for the same period will create duplicates. Disconnect or pause the bank feed first, or import only the date range not covered by the feed.

Frequently Asked Questions

What file format does Wave use for bank statement imports?

Wave accepts CSV files only for manual bank imports. The file must have columns: Date, Description, Amount. Wave does not accept OFX or QBO files.

How do I import a PDF bank statement into Wave?

Wave cannot read PDFs. Convert the PDF to CSV first using Zera Books or a similar tool, then import via Accounting > Transactions > Import.

Does Wave have a duplicate transaction check on CSV import?

Wave does check for duplicate transactions on import but the matching is basic — it compares date and amount. Transactions with the same date and amount but different descriptions may still be flagged. Review the import preview carefully.
