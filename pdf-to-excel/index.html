Pre-Formatted Import Templates

In addition to the standard XLSX output, Zera Books offers pre-formatted XLSX templates optimized for direct import into specific accounting platforms.

QuickBooks Online Template

Columns mapped to QBO's bank transaction import format: Date, Description, Amount (signed), Account. Includes category mapped to QBO expense account names.

Xero Template

Columns: Date, Amount, Payee, Description, Reference, Account Code. Amounts are signed (positive = deposit, negative = withdrawal). Compatible with Xero's manual import.

Sage 50/Intacct Template

Mapped to Sage journal entry import format with GL account codes. Date format set to YYYY-MM-DD per Sage requirements. Debit/credit split maintained.

Generic Pivot-Ready Format

All transactions in a single table with named columns. No merged cells, no header block — designed for immediate pivot table analysis in Excel or Google Sheets.

Wave Accounting Template

Columns: Date, Description, Amount (signed), Account Name. Compatible with Wave's CSV/Excel import wizard. Negative values for expenses, positive for income.

Zoho Books Template

Mapped to Zoho's transaction import format. Includes vendor/customer name columns when available. Date format: MM/DD/YYYY per Zoho requirements.

Frequently Asked Questions

What Excel version format does Zera Books export to?

Zera Books exports to .xlsx format (Excel 2007 and later). The file is compatible with Microsoft Excel 2007+, Google Sheets, LibreOffice Calc, and Apple Numbers. Legacy .xls format is not available.

How are dates formatted in the Excel output?

Dates are formatted as Excel date serial numbers with the cell format set to MM/DD/YYYY by default. Users can change the date format in Excel without affecting the underlying data. The format is also configurable in Zera Books export settings (MM/DD/YYYY, DD/MM/YYYY, YYYY-MM-DD).

Does the Excel output include formulas or just values?

The standard output contains values only (no formulas). The QuickBooks-ready template includes SUM formulas in the totals row. Users can request a formula-enabled version which includes running balance formulas in the Balance column.

Can Zera Books export multiple accounts to separate worksheets in one file?

Yes. When multi-account detection identifies multiple accounts, users can choose to either export separate files (one per account) or a single workbook with each account on its own worksheet. The worksheet is named with the account type and last 4 digits.
