CSV to QBO Converter: Technical Reference for QuickBooks Import

Banks often provide transaction downloads in CSV format. QuickBooks Online doesn't accept CSV directly for bank imports — it requires QBO (Open Financial Exchange) format. Converting between them involves mapping CSV columns to the QBO XML structure and generating required fields like FITID that don't exist in a raw CSV.

QBO File Structure

A valid QBO file has three sections:

  • Header — file creation date, version, and account information
  • BANKTRANLIST — the list of transactions with date range
  • LEDGERBAL — closing balance (required by some QBO validators)

Each transaction in the BANKTRANLIST must include TRNTYPE, DTPOSTED, TRNAMT, FITID, and NAME. MEMO is optional but improves matching in QuickBooks.

FITID Generation

FITID (Financial Institution Transaction ID) must be unique per transaction. CSV files don't include FITIDs. The converter generates them by hashing the date, amount, and description — so the same transaction always produces the same FITID, which prevents duplicate imports if you upload the same CSV twice.

Common rejection reason: QuickBooks rejects QBO files where any FITID is blank or duplicated within the file. Always use a converter that generates proper FITIDs rather than sequential numbers, which don't guarantee uniqueness across multiple uploads.

Handling Debit and Credit Columns

Many bank CSVs use separate Debit and Credit columns rather than a single signed Amount column. Before conversion, these need to be combined: debit amounts become negative, credit amounts stay positive. Zera Books handles this automatically when it detects the two-column pattern.

Frequently Asked Questions

What columns does a CSV need to convert to QBO?

At minimum: Date, Amount, and Description. The converter generates FITID, DTPOSTED formatting, and TRNTYPE automatically.

Why does QuickBooks reject my QBO file?

Most common causes: incorrect DTPOSTED format (must be YYYYMMDD), missing or duplicate FITIDs, or special characters in description fields.

Can I convert a CSV from any bank to QBO format?

Yes. Zera Books maps date, amount, and description columns to QBO fields regardless of the source bank's column naming.
