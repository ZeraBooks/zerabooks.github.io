Converting Bank Statements to IIF for QuickBooks Desktop

QuickBooks Desktop users can't use QBO files — that format is specific to QuickBooks Online. The Desktop versions (Pro, Premier, Enterprise) use IIF (Intuit Interchange Format) for transaction imports. If you're converting bank statement PDFs for a client on QuickBooks Desktop, IIF is the target format.

IIF File Structure

IIF is a tab-delimited text format with type headers that tell QuickBooks what kind of data each section contains. A bank transaction import uses the TRNS (transaction) and SPL (split) record types:

!TRNS	TRNSID	TRNSTYPE	DATE	ACCNT	NAME	AMOUNT	MEMO
!SPL	SPLID	TRNSTYPE	DATE	ACCNT	AMOUNT	MEMO
!ENDTRNS
TRNS	1	BANK	01/15/2025	Checking	Vendor Name	-247.50	Office supplies
SPL	2	BANK	01/15/2025	Uncategorized Expense	247.50	Office supplies
ENDTRNS

Each transaction in IIF requires a TRNS line (the bank account side) and at least one SPL line (the offsetting account). Zera Books generates both, using "Uncategorized Expense" or "Uncategorized Income" as the offset account unless AI categorization is enabled, in which case the assigned category is used.

Account Name Matching

Important: The ACCNT field in the IIF file must exactly match the account name in your QuickBooks Desktop chart of accounts. If you use "Business Checking" in QuickBooks but the IIF says "Checking", the import will fail or create a new account. Check your exact account names before generating the IIF file.

Date Format in IIF

IIF uses MM/DD/YYYY date format. This differs from QBO (YYYYMMDD) and OFX. Zera Books handles the date conversion automatically, but if you're building IIF files manually, ensure the date format is correct — QuickBooks Desktop is strict about this.

Importing the IIF File

  1. Back up your QuickBooks Desktop company file first
  2. Go to File → Utilities → Import → IIF Files
  3. Select the .iif file from Zera Books
  4. Click Open — QuickBooks processes the file immediately
  5. Review the imported transactions in the bank account register

Unlike QBO imports in QuickBooks Online, IIF imports in QuickBooks Desktop don't provide a review step before committing. The transactions are added directly. This is why backing up first is important.

Frequently Asked Questions

What is IIF format in QuickBooks?

IIF (Intuit Interchange Format) is a tab-delimited text format used by QuickBooks Desktop for importing transactions and lists. It's the primary import format for QuickBooks Pro, Premier, and Enterprise.

How do I import an IIF file into QuickBooks Desktop?

File → Utilities → Import → IIF Files. Select the file and click Open. Back up your company file first — IIF imports commit immediately without a review step.

What is the difference between IIF and QBO for QuickBooks?

IIF is for QuickBooks Desktop. QBO is for QuickBooks Online. They use different formats and are not interchangeable.
