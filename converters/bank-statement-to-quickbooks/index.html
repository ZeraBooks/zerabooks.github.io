Converting Bank Statement PDFs to QuickBooks QBO Format

QuickBooks Online's bank feed requires transactions in QBO format. If your bank supports direct connection, QBO files are generated automatically. But for older institutions, foreign banks, or historical statements going back beyond what the live feed covers, you need to convert the PDF statement to a QBO file manually.

What the QBO Format Contains

QBO is an XML-based format developed by Intuit. Each transaction record includes:

  • DTPOSTED — transaction date in YYYYMMDD format
  • TRNAMT — amount (negative for debits, positive for credits)
  • FITID — unique transaction ID to prevent duplicate imports
  • NAME — payee or description
  • MEMO — additional description if present
  • TRNTYPE — transaction type (DEBIT, CREDIT, CHECK, etc.)

The file also contains account information (routing number, account number, account type) and a date range header. QuickBooks reads all of these fields when importing.

Importing the QBO File into QuickBooks Online

  1. In QuickBooks Online, go to Banking → Banking
  2. Click Upload transactions (top right)
  3. Select Upload from file
  4. Choose the .qbo file from Zera Books
  5. Select the account to import into
  6. Review the transaction list and confirm

QuickBooks automatically skips any transactions with FITIDs it has already imported, so uploading overlapping date ranges won't create duplicates.
